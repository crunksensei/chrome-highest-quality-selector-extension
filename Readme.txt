# Max Quality Chrome Extension

## Introduction
The Max Quality Chrome Extension is designed to enhance your video streaming experience by automatically selecting the highest available quality for video content across various platforms. Say goodbye to manually adjusting settings every time you watch a video. With this extension, you can enjoy the best viewing experience without interruption.

## Installation Instructions
To install the Max Quality Chrome Extension, follow these simple steps:

1. Download the extension package.
2. Unzip the downloaded file to a folder on your computer.
3. Open Chrome and navigate to `chrome://extensions/`.
4. Enable Developer Mode by toggling the switch in the top-right corner.
5. Click on the "Load unpacked" button.
6. Navigate to the folder where you unzipped the extension, select it, and click "Open".
7. The extension is now installed! You will see the Max Quality Chrome Extension icon in your browser's extension toolbar.

## How to Use
Once installed, the extension will automatically start working. There is no need for any manual setup. Just start watching videos, and the extension will take care of the rest.
